# Coral-Sparkles---Skin-for-ao3
A soft watermelon/coral themed AO3 site skin with glitter kudos messages.  Features: • watermelon gradient header • pastel tag styling • glowing logo • custom glitter kudos message • softer UI cards

📦 Installation
1. Log in to Archive of Our Own
2. Go to Dashboard
3. Click Skins
4. Click Create Site Skin
5. Give your skin a name
6. Paste the code into the CSS field
7. Save the skin
8. Click Use

Your AO3 interface should now use the Watermelon Stardust theme.
